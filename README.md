# Valentine's Day Interactive Site

A cute, interactive Valentine's Day website where the "No" button is impossible to click!

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
